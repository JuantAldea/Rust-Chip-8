# Rust-CHIP-8 Interpreter
Just another CHIP-8 interpreter written in Rust.

## Missing features
* Sound.
* Proper timers.

## Compile & Run
```$ cargo run --release```
## Controls
CHIP-8 Machines have an hexadecimal pad, which is mapped into standard QWERTY like so:

```
HEX PAD | QWERTY
1 2 3 C | 1 2 3 4
4 5 6 D | Q W E R
7 8 9 E | A S D F
A 0 B F | Z X C V
```
